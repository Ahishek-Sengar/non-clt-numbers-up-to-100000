# non-clt-numbers-up-to-100000
This repository contains computational data related to CLT and Non-CLT numbers obtained using Berger’s criterion.
The datasets include the complete lists of Non-CLT numbers and CLT numbers in the range 2 ≤ n ≤ 100000.
